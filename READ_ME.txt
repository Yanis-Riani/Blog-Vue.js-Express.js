Fonctionnalité:

	Front:
		- Consulter tous les articles du site
		- Consulter les articles d’une catégorie en particulier
		- Faire des recherches sur le site
		- Se connecter a/créer un compte
		- Sauvegarder un article en favoris
		- Espace dédié aux administrateur
		- Créer de nouveaux articles
	Back: 
		- Une api renvoyant des données en JSON
		- Des routes permettant de créer des articles, utilisateurs, commentaires, catégorie
		- Gestion des likes et des favories 
		- Implémentation des principe REST
		- Sytem d'authentification
		- Gestion des permissions

Commande:

	Lancement:
		npm run start (back end)
		npm run serve -- --port 3000 (front end)
